Values in Scala

  • Keyword val declares a value of a type
  • val name: Type = expression
    • val name: String = "Jacek"
val n = 5
val sum100 = 1 to 100 sum
  • Pattern matching on assignment
    • val (x, y) = (3, 5)
val x :: xs = List(1, 2, 3, 4)
